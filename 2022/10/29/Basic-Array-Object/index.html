Basic - Array, Object

Array

While we use const to declare the Array, we still can add/remove the items inside.

const fruits = ["Apple", "Banana", "Grape"];

fruits.push("Orange");
// output: ["Apple", "Banana", "Grape", "Orange"];

fruits.split(-1);
// output: ["Apple", "Banana", "Grape"];

Because the Arrayâ€™s value store in Heap memory, we can change it with any methods.
But, we cannot Reassign it!

fruits = ["Orange"];
// TypeError: Assignment to constant variable.

Use [index] to access items of Array.

fruits[0];
// output: Apple

Object

Object works the same with it. The value store in Heap.

const person = { name: "Leeso", age: 30 };

person.job = "Dev";
// output: {name: 'Leeso', age: 30, job: 'Dev'}

person = { name: "Leeso", age: 30, job: "Dev" };
// TypeError: Assignment to constant variable.

Use . or [expression] to access/create properties of Object.

const person = { name: "Leeso", age: 30, job: "Dev" };

person.name;
// output: 'Leeso'

person["a" + "g" + "e"];
// output: 30

const str = "job";
person[str];
// output: Dev

person["twitter"] = "@lesso";
person.home = "Taipei";
// output: { name: "Leeso", age: 30, job: "Dev", twitter: '@leeso', home:'Taipei'};

this keyword in object, always point to who calling it.

const leeso = {
  name: "leeso",
  age: 30,
  sayhi() {
    console.log(`Hi, I am ${this.name}, ${this.age} years old.`);
  },
};

leeso.sayhi();
// output: Hi, I am leeso, 30 years old.

Use this.property to create new property.

const leeso = {
  name: "leeso",
  age: 30,
  birthYear() {
    this.birthYear = new Date().getFullYear() - this.age;
  },
};

leeso.birthYear();
// output: {name: 'leeso', age: 30, birthYear: 1992}
